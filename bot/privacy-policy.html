furry bot - privacy policy

last updated: june 2026

overview

this privacy policy describes how furry bot ("the bot", "we") handles data when you use it on Discord. we believe in minimal data collection and full transparency.

what we collect

furry bot collects only the following:

  • anonymized command counts - a counter is incremented each time any user runs a command. this count is a bare number and is not associated with any user ID, username, server, or any other identifying information.

what we do not collect

  • user IDs, usernames, or discriminators
  • server (guild) IDs or names
  • message content or command arguments
  • IP addresses or device information
  • any personally identifiable information

how data is used

the anonymized command count is used solely to understand general usage volume and improve the bot. it cannot be used to identify or track any individual user.

data sharing

we do not sell, rent, or share any collected data with third parties. the anonymized count data may be displayed publicly (e.g. on a stats page) but contains no personal or identifiable information.

data retention

command count data is retained indefinitely as it contains no personal information. since no user-linked data is collected, there is nothing to request deletion of.
